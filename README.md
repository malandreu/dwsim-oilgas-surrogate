# Surrogate Modelling of an Oil & Gas Separation Plant

Reproducing and extending Andreasen (2020) using DWSIM v9 and a modern Python stack.

## Overview

This project builds a surrogate model (Gaussian Process Regression / Kriging) for a 
North Sea crude oil stabilisation plant simulated in DWSIM. The surrogate replicates 
the rigorous simulator with R² = 0.9998 and evaluates ~100,000× faster, enabling 
practical multi-objective optimisation with NSGA-II.

![DWSIM oil & gas separation plant flowsheet](figures/dwsim_oil_gas_flowsheet.png)

**Optimisation results** (vs design base case from Andreasen 2022, Table 1):
- Crude oil recovery: **+0.72%** (+20 kmol/h)
- Total compression power: **−3.36%** (−175 kW)
- Both objectives improve simultaneously — no trade-off required

## Workflow
```
DWSIM v9 → 100 LHS simulations → GPR surrogate (R²=0.9998) → NSGA-II → optimal conditions
```

## Stack

| Tool | Purpose |
|---|---|
| DWSIM v9 (open source) | Rigorous process simulation |
| win32com + pythonnet | Python automation of DWSIM |
| scipy LHS | Design of experiments |
| scikit-learn GPR | Surrogate model (Kriging with ARD) |
| pymoo NSGA-II | Multi-objective optimisation |

## Repository structure
```
notebooks/   — Jupyter notebook with full workflow
data/        — Generated dataset (100 LHS simulations)
figures/     — All plots generated by the notebook
```

## Usage

1. Install DWSIM v9 from https://dwsim.org
2. Clone this repository
3. Install Python dependencies:
```bash
   pip install -r requirements.txt
```
4. Edit the path configuration cell in the notebook
5. Run `notebooks/DWSIM_OilGas_Surrogate.ipynb`

> **Windows only.** The COM interface used to automate DWSIM (`pywin32`) is not
> available on Linux or macOS.
